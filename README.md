# Preparation
## Requirements

- Python 3.7+
- PyTorch 1.10.0+
- PyTorch Geometric 2.1.0+


## Datasets

The datasets should be organized as:

```
.
├── data
│   └── ogb
│   └── coauthor
│   └── ...
│       
```

# Getting Started

- Train with command line

```bash
python train.py --cuda_num 'your chosen cuda' --nbatch 1 --testmode 'your output folder/' --dataset 'dataset name'
```
